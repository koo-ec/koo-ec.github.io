# Privacy

This academic website is hosted through GitHub Pages and is maintained by Dr Koorosh Aslansefat.

Google Analytics is used only after a visitor gives consent. When enabled, analytics helps understand page visits, navigation, downloads, outbound links, and site searches so the website can be improved.

The website does not intentionally send names, email addresses, or other directly identifying personal details to Google Analytics.

You can reject analytics when the cookie-consent message appears. If you accept analytics and later change your mind, use the **Change cookie settings** link in the footer to withdraw or update your choice.

For website questions, use the contact details on the [Contact](contact.md) page.
