# KTP: Connexin

This Knowledge Transfer Partnership works with Connexin on technology for adult social care. The project centres on a speech-to-text platform that brings sensor-network information into a smart-speaker environment for assisted living.

## Project Details

| Item | Detail |
| --- | --- |

| Value | £181,302 |
| Dates | 1 October 2023 - 30 September 2025 |
| Partner | Connexin |
| Theme | Energy, Environment and Sustainability |
| Koorosh's role | Co-Investigator |

## Project Focus

- Integrating speech interfaces with assisted-living sensor networks.
- Designing AI-enabled support for adult social care settings.
- Connecting IoT data, voice interaction, and dependable system behaviour.
- Supporting technology transfer between university research and industrial deployment.
