# Alan Turing Institute Post-Doctoral Enrichment Award

This award supported postdoctoral activity connected to Koorosh Aslansefat's work on SafeML and machine-learning model monitoring. It sits within a broader programme intended to help early-career researchers develop collaborations, skills, and independence in data science and AI.

## Project Details

| Item | Detail |
| --- | --- |
| Type | Postdoctoral enrichment award |
| Status | Complete |
| Funder | The Alan Turing Institute |
| Value | £2,000 |
| Dates | 1 March 2022 - 31 October 2022 |
| Project area | SafeML, uncertainty quantification, and ML safety monitoring |
| Koorosh's role | Awardee |

## Project Focus

- Developing SafeML as a practical approach to machine-learning safety monitoring.
- Strengthening research links around dependable and trustworthy AI.
- Supporting publication, collaboration, and postdoctoral research development.
