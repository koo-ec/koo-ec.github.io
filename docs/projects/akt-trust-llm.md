# AKT TRUST-LLM

**Tailored Responsible Use of Safe and Trustworthy LLM for Planning Law Institutional Memory Management**

TRUST-LLM developed a secure institutional-memory concept for planning-law practice. The project focused on combining a specialised large language model with knowledge-graph support so that complex legal material can be searched, reused, and interpreted with stronger attention to privacy, accuracy, and regulatory compliance.

## Project Focus

- Responsible use of LLMs in legal and planning-law workflows.
- Institutional memory management for complex case material.
- Knowledge-graph support for traceable retrieval and contextual search.
- Practical safeguards for privacy, accuracy, and trustworthy deployment.
